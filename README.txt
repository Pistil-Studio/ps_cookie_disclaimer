#typo3-ps_cookie_disclaimer

Typo3 Compatibility : 4.X (Just install the extension to add the new fonctionnality)

Configuration (FR)
Page Id with explanation :
Ceci est l’identifiant de la page créée contenant les informations sur la politique des cookies.
Si cette valeur est renseignée ou différentes de 0, le lien « CNIL link Label » sera rajouté automatiquement à la suite du message du bandeau de la CNIL, avec un lien pointant sur cette fameuse page.
La page doit être créer sous la page racine : « Espace Assurés - APRIL Assurances Entreprises » pour hériter de son « gabarit »

Activate CNIL acceptation
Permet d’afficher le bandeau ou non en cochant la case :
Cochée : le bandeau est affiché
Décochée : le bandeau est masqué

CNIL Content :
Permet de changer le texte affiché dans le bandeau

CNIL Link Label :
Permet de changer le libellé du lien pointant sur la page de détail
